# 📊 Day 5 – Sales Performance Dashboard (Microsoft Excel)

## 📌 Overview

On Day 5 of my 90-day Data Analyst learning journey, I built an interactive Sales Performance Dashboard using Microsoft Excel.

This project combines Pivot Tables, Pivot Charts, KPIs, Slicers, and Business Insights to analyze sales performance using a retail sales dataset.

---

## 🎯 Objectives

- Build business KPIs
- Create Pivot Charts
- Design an interactive dashboard
- Analyze sales performance
- Practice business storytelling using data

---

## 📈 Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Average Sales
- Highest Sale
- Lowest Sale
- Number of Orders
- Number of Customers

---

## 📊 Dashboard Components

- Sales by Region
- Sales by Category
- Yearly Sales Trend
- Sales Distribution by Category
- Region Slicer
- Category Slicer

---

## 💡 Business Insights

- West region generated the highest sales.
- Office Supplies contributed 62% of total sales.
- Technology contributed the smallest share of sales.
- Sales peaked in 2023.
- Total sales exceeded **$2.3 million** across **10,194 orders**.
