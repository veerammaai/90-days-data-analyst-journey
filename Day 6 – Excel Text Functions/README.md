# Day-6 – Excel Text Functions

## 📌 Objective

Today's goal was to learn how Excel Text Functions help clean, standardize, and transform raw business data before analysis.

---

## 📚 Topics Covered

- LEFT()
- RIGHT()
- MID()
- LEN()
- TRIM()
- UPPER()
- LOWER()
- PROPER()
- CONCAT()

---

## 💻 Practice Completed

Using the Superstore Sales dataset, I practiced:

- Extracting Order Prefix
- Extracting Order Number
- Extracting Order Year
- Finding Customer Name Length
- Converting text to UPPERCASE
- Converting text to lowercase
- Formatting names using PROPER()
- Removing unnecessary spaces using TRIM()
- Combining First Name and Last Name using CONCAT()

---

## 💡 Business Insights

- Customer names should follow a consistent format to maintain a clean customer database.
- Extra spaces in text fields can lead to incorrect lookups and duplicate records.
- Order IDs often contain meaningful information such as region codes and order numbers.
- Text functions simplify data preparation, reducing manual effort and improving reporting accuracy.
- Clean and standardized text data is essential before performing lookups, Pivot Tables, dashboards, or importing data into Power BI or SQL.

---

## 🎯 Key Learning

Text Functions are essential for preparing raw data before analysis. Properly cleaned and standardized data improves reporting accuracy, supports better business decisions, and creates a reliable foundation for dashboards and advanced analytics.
