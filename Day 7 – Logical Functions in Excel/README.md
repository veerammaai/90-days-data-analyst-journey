# Day 7 – Logical Functions in Excel

## Overview

On Day 7, I practiced Excel's logical functions using a retail sales dataset. The goal was to understand how businesses use conditional logic to classify data, identify important records, and automate decision-making.

## Topics Covered

* IF Function
* AND Function
* OR Function
* NOT Function

## Practice Scenarios

* Classified sales as **High Sale** or **Regular Sale**
* Identified orders as **Profit** or **Loss**
* Evaluated sales performance based on defined conditions
* Combined multiple conditions using **AND**
* Flagged records requiring review using **OR**
* Used **NOT** to identify records outside a specified region

## Key Takeaways

* Learned how logical functions support business decision-making.
* Gained confidence in combining multiple conditions within formulas.
* Understood how conditional logic is commonly used in sales and reporting dashboards.
* Continued improving practical Excel skills using a real-world dataset.
